# Kelvin Mak Portfolio (Static HTML)

This repository contains a static HTML/CSS/JS portfolio site and is ready for GitHub Pages deployment.

## Publish on GitHub Pages

1. Create a new GitHub repository and push this project.
2. Ensure your default branch is `main`.
3. In GitHub, open `Settings` -> `Pages`.
4. Under **Build and deployment**, set **Source** to `GitHub Actions`.
5. Push to `main` (or re-run the workflow) and wait for deployment.
6. Your site will be available at:
   - `https://<your-username>.github.io/<your-repo-name>/`

## Local preview

Open `index.html` directly in your browser, or use a local static server.
